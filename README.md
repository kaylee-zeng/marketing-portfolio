# Kaylee Zeng — Marketing Portfolio

Global Marketing Manager | B2B Marketing · Brand · Growth

A personal marketing portfolio website showcasing B2B marketing case studies, capabilities, and achievements.

## Design

Apple-minimalist single-page design with smooth scroll animations, built with pure HTML/CSS/JS — zero dependencies.

## Sections

- **Hero** — Name, title, and introduction
- **About Me** — Background, languages, and marketing funnel visualization
- **Marketing Capabilities** — 6 core skills with star ratings
- **Tech Stack** — CRM, analytics, AI tools, design, platforms
- **Case Studies** — 5 detailed B2B marketing cases with strategy flows and results
- **AI-Driven Marketing** — AI tools and automation expertise
- **Contact** — Email and WeChat

## Tech

- HTML5, CSS3, vanilla JavaScript
- CSS custom properties for theming
- Intersection Observer for scroll animations
- Fully responsive (mobile, tablet, desktop)

## Deploy

This site is deployed via GitHub Pages. To deploy your own:

1. Push to a GitHub repository
2. Go to Settings → Pages
3. Set source to the branch containing these files
4. Save — your site will be live at `https://<username>.github.io/<repo>/`
